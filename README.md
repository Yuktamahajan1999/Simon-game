# Simon-game
# Overview :
- Simon Game is an interactive memory game built using HTML, CSS, and JavaScript.
- The game challenges players to repeat an ever-increasing sequence of light and sound patterns.
- The objective is simple: follow the sequence correctly to keep the game going.
- As the sequence lengthens, the game tests your memory and reflexes, making it a fun and engaging challenge for all ages.

# Features :
- **Memory Challenge :** The game presents a sequence of colors and sounds that the player must repeat. Each round, the sequence becomes longer.<br>
- **Color and Sound Feedback :** Each of the four buttons (colored in red, green, blue, and yellow) lights up and plays a unique sound, providing clear visual and audio.<br>
- **Difficulty Progression :** As the player successfully repeats the sequence, the game becomes progressively harder with longer sequences to remember.<br>
- **Game Over :** The game ends when the player makes an incorrect move. The player can restart the game at any time.<br>
- **Responsive Design :** The game’s layout is responsive and adapts to different screen sizes, allowing for a smooth experience on both desktop and mobile devices.<br>
# Technologies Used :
- **HTML :** For structuring the game’s interface, including buttons, score display, and other elements.<br>
- **CSS :** For styling the game’s layout, buttons, and visual feedback (e.g., highlighting buttons when they are clicked or lit up).<br>
- **JavaScript :** For implementing the game logic, including generating random sequences, handling user input, and comparing the player’s responses to the sequence.<br>
# How to Play :
- **Start the Game :** Click the "Start" button to begin.
- **Follow the Sequence :** The game will show you a sequence of colors and play corresponding sounds. Watch and listen carefully!
- **Repeat the Sequence :** Click the colored buttons in the same order as the sequence you were shown.
- **Game Progression :** If you get the sequence right, a new color will be added to the sequence, and the game continues. The sequence becomes progressively longer and faster.
- **Lose the Game :** If you make a mistake, the game ends, and you will be given the option to restart.
- **Restart :** You can always restart the game by clicking the "Start" button again.
